# Oráculos

Para o funcionamento desses mercados, é necessário ter uma maneira confiável de definir o preço dos ativos, ainda que a maior parte do volume esteja em mercados externos à rede, como as exchanges centralizadas.

Para isso, os protocolos usam oráculos, sendo o maior deles o protocolo Chainlink, uma rede decentralizada que fornece dados confiáveis através de múltiplos nós independentes e agregação criptográfica.

Por utilizar vários nós, limita o risco de que o comprometimento de uma fonte gere maiores problemas. Além disso, existe um modelo de reputação, staking e recompensas. Nós que querem participar da rede devem depositar $LINK em gerantia. Se fornecerem dados corretos, são pagos em $LINK e ganham reputação. Dados imprecisos podem levar aos $LINK depositados em garantia a serem cortados/confiscados, além da perda de reputação.
